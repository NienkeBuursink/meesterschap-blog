Websites

  1. Sprint 0

    Aan het begin van de minor wist ik nog niet goed wat je allemaal kon met HTML, CSS en JavaScript. Tijdens dit project met de gitaar-website heb ik geleerd dat je met vooral CSS al heel veel coole en interactieve dingen kunt maken, zoals vormen, animaties en effecten.

    Ook merkte ik dat ik al snel meer leerde door te experimenteren met nieuwe dingen zoals API’s, en view transitions. Niet alles werkte meteen, maar daardoor leerde ik juist veel.

    Wat ik meeneem is dat je vooral door te proberen en fouten te maken snel vooruitgaat, en dat webdevelopment veel creatiever is dan ik eerst dacht.

  2. Browser Technologies

    Tijdens dit project heb ik gewerkt aan een formulier voor aangifte erfbelasting. Ik heb geleerd dat je met HTML en CSS al een heel sterk en logisch formulier kunt bouwen, en dat JavaScript vooral gebruikt wordt om het slimmer en gebruiksvriendelijker te maken.

    Wat ik vooral heb meegenomen is hoe belangrijk het is om goed na te denken over UX, zoals progressive disclosure, validatie en duidelijke inputvelden. Dingen als BSN-validatie, local storage en het dynamisch toevoegen van erfgenamen waren soms best lastig, maar gaven wel veel inzicht in hoe formulieren in de praktijk echt werken.

  3. CSS

    Tijdens dit project heb ik een 3D Rubiks kubus gemaakt met CSS, en daarin veel geleerd over wat er allemaal mogelijk is met 3D in de browser. In het begin was het vooral veel uitproberen en zelfs een beetje koppig mijn eigen aanpak blijven volgen, maar uiteindelijk heb ik geleerd dat sommige problemen gewoon een hele andere kijk nodig hebben.

    Wat ik vooral heb meegenomen is dat CSS veel krachtiger is dan ik eerst dacht, zeker als je met 3D transforms en animaties werkt. Ook heb ik geleerd hoe belangrijk het is om goed te luisteren naar feedback en artikelen te gebruiken als je vastloopt, want daardoor kon ik uiteindelijk mijn rotaties en effecten echt goed werkend krijgen.

    Daarnaast vond ik het interessant om te zien hoe kleine keuzes in structuur (zoals extra containers of positionering) grote invloed hebben op het eindresultaat. Uiteindelijk heb ik nu veel meer begrip van 3D CSS en wil ik daar later zeker nog verder mee experimenteren, misschien zelfs met tools zoals three.js.

  4. Hackathon

    Deze website heb ik met een groep van vier personen gemaakt, met een externe opdrachtgever. Het is een website over de Nebula Explorer, een satelliet die in 2030 de ruimte in wordt gelanceerd. Tijdens dit project heb ik goed leren communiceren met mijn groepsgenoten en heb ik mijn aandeel in zo'n project ontdekt.
